# Evaluation Rubric (DSP/CE Tutor)
Date: 2025-10-18

## Automatic checks
- Numeric: abs or relative tolerance; confirm units when specified.
- Multiple-choice: exact match.
- Short-text: keyword/phrase match; manual review recommended for pedagogy.
- (Optional) Code: execute snippets; compare shape/value.

## Human pedagogy audit (0–2 each unless stated)
- Correctness
- Clarity & structure
- Unit usage & assumptions (0–1)
- Step-by-step reasoning
- Code quality & comments

## Passing criteria
- ≥80% numeric within tolerance, ≥60% short-text correct, multiple-choice ≥80%,
- Human mean ≥6/9 on 20-sample audit.
